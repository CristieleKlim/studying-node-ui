# studying-node-ui

IONIC http://ionicframework.com/ 

Ionic CLI and Cordova
$ npm install -g ionic
$ npm install -g cordova

* Linux : Necessário utilizar sudo

Create first Ionic app
$ ionic start NAME --v2

Run app
$ ionic serve

Run lab mode (http:// … :8100/ionic-lab)
$ ionic serve -l 

Generate new page
$ ionic g page myPage
